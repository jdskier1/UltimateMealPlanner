# Meal Picker App

This project turns the spreadsheet into a Streamlit app, with the **Meal Selector** sheet translated into the main UI.

## Run locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Included features

- 7-day meal selector for breakfast, lunch, and dinner
- spreadsheet-style random fill controls
- grocery list generation from selected meals
- calendar CSV export
- searchable meal library
