Meal file template

How to use:
1. Edit meal.json with the meal details.
2. Put your photo files into the folders referenced by meal.json.
3. Keep the paths in meal.json matched to the files inside this zip.
4. Zip the full contents and import it from the Meal Library > Import Meals button.

Supported structure:
- meal.json
- meal_photos/...
- ingredient_photos/<ingredient_name>/...
- step_photos/step_1/...

You can add custom columns as extra top-level fields in meal.json.
